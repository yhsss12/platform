# 工作台组内受控试用说明

当前版本为**组内受控试用版（Internal Alpha）**，面向小范围同学验证主流程与稳定性，尚未开放为个人独立工作空间产品。

## 环境说明

- **共享实验环境**：所有人登录后可见同一批数据集、训练任务、评测任务与模型资产。
- **暂不做个人隔离**：未启用 user owner 隔离与 workspace 独立资源隔离；请勿删除他人创建的数据、任务或模型。
- **运行节点**：训练与评测依赖管理员配置的 GPU / 仿真节点；部分能力在当前环境可能显示「暂未开放」。

## 推荐试用主流程

建议按以下顺序体验线缆穿杆（Cable Threading）闭环：

1. **任务模板** → 选择「线缆穿杆」→ **生成数据**
2. **数据中心** → 查看生成的数据集与 **Expert Rollout** 回放
3. **训练中心** → 选择上述数据集，训练 **ACT** 或 **Robomimic BC**
4. **模型资产** → 查看训练产出的 **Final checkpoint**
5. **评测中心** → 选择模型资产 **启动模型评测**
6. **回放** → 查看 **Model Rollout**（与 Expert Rollout 区分）

## 暂不建议普通同学自行使用

以下能力仍在实验或依赖专用环境，组内试用期间请避免作为主流程：

- Isaac Lab 新建数据生成
- 双臂线缆数据生成
- pi0 / DT / LeRobot 等尚未开放的训练后端
- 大规模并行训练或数据生成

如需验证上述能力，请先与管理员确认环境是否就绪。

## 反馈方式

遇到问题或建议时，请提供以下信息以便排查：

| 字段 | 说明 |
|------|------|
| 账号 | 登录用户名 |
| 页面 | 出问题的 URL 或菜单路径 |
| jobId | 相关训练 / 评测 / 数据生成任务 ID |
| 操作步骤 | 从哪一步开始、点了什么 |
| 截图 | 错误界面或异常状态 |
| 期望结果 | 你认为应该怎样 |
| 实际结果 | 实际发生了什么 |

## 页面与能力边界

- **数据中心 / 训练中心 / 评测中心 / 模型资产 / 任务模板**：组内试用主入口。
- **过程评测详情**（`/workspace/evaluation/process`）：内部实验页面，已从导航与业务详情断开，组内试用不开放。
- **实验页**（`/workspace/experiments`）：不在主导航中，不作为试用主流程。

## 登录

请使用管理员分配的组内试用账号登录。若无法登录或权限异常，请联系管理员。
