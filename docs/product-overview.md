# 一湃智能数据平台 - 产品概述

本文档从产品视角介绍一湃智能数据平台的定位、用户角色与功能模块，便于产品、运营与客户理解 v0.1 的能力边界。

---

## 1. 产品定位

**一湃智能数据平台**（Yipai Intelligent Data Platform）面向**机器人、自动驾驶与智能装备**的数据闭环需求，提供从**数据采集、存储、标注、转换**到**资产管理与协作**的一站式能力。  
目标用户包括算法团队、数据标注团队、项目管理员与系统管理员；v0.1 聚焦核心链路可用，为后续数据清洗、高级分析与策略下发预留扩展点。

---

## 2. 用户角色

| 角色 | 说明 | 典型能力 |
|------|------|----------|
| **管理员** | 平台级管理 | 用户管理、审计日志、全部项目与设备 |
| **负责人** | 项目与资源管理 | 项目管理、成员、设备、采集/标注/转换入口 |
| **普通成员** | 日常使用 | 数据资产查看、采集任务执行、标注、转换（受项目与权限约束） |

- 未登录用户仅可访问登录页；登录后根据角色展示不同侧栏与入口。
- 未加入某项目的用户不可见该项目下的任务与资产（列表与详情均受控）。

---

## 3. 功能模块总览

```
┌─────────────────────────────────────────────────────────────────┐
│                    一湃智能数据平台 v0.1                          │
├─────────────────────────────────────────────────────────────────┤
│  概览 │ 数据资产 │ 采集（任务/作业/实时/质量）│ 标注 │ 转换 │ 管理  │
├─────────────────────────────────────────────────────────────────┤
│  登录/权限  │  中英多语言  │  后台任务面板  │  统一弹窗/Toast   │
└─────────────────────────────────────────────────────────────────┘
```

- **概览**：数据与任务统计、活动流、项目分布、告警区。
- **数据资产**：数据列表、筛选、导入（HDF5/MCAP/LeRobot）、导出、跳转标注/转换。
- **采集**：任务管理、作业中心、实时采集、质量校验。
- **标注**：标注任务、执行页（Episode/帧/描述）、自动描述（OpenAI 兼容 API）、模型管理。
- **转换**：按项目选择资产、新建转换、进度与日志（当前前端 Mock）。
- **管理**：项目、设备、用户（超管）、审计日志（超管）。

---

## 4. 核心业务流程简述

1. **数据接入**：在「数据」页导入 HDF5/MCAP/LeRobot 等，或通过采集任务产生数据并登记为资产。
2. **采集**：创建采集任务（绑定项目、设备、存储路径等）→ 作业中心认领 → 实时采集或质量校验 → 数据落盘并可与数据资产关联。
3. **标注**：创建标注任务（选项目与数据集）→ 进入执行页 → 选择 Episode、编辑/自动生成任务描述 → 保存或批量下载。
4. **转换**：在转换页按项目选择数据资产，创建转换任务，查看运行进度与结果（后端流水线可扩展）。
5. **协作**：通过项目与成员管理控制可见范围；通过后台任务面板统一查看采集/标注/转换/导出进度。
